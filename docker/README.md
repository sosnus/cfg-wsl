## Basic commands

### How to see logs from container 
* `docker logs -f nginx-container`

### How to connect to container
* `docker exec -it nginx-container /bin/bash`


### Basic flags
* `-it` - interactive
* `-d` - headless
* `--restart=always` - run after restart
